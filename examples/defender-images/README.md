# Defender Example Images

This directory is reserved for a bounded example-image corpus for the public repository.

The intention is to place a curated subset of Defender images here as examples of the visual material used to train the models, without distributing the model checkpoints themselves and without shipping the original 3D source asset.

## Expected Contents

- selected example image files
- [`NOTICE.md`](NOTICE.md) with attribution and licensing context
- [`ZIP_NOTICE.txt`](ZIP_NOTICE.txt) for inclusion in any zipped distribution of the same image subset

## Scope

This directory is for a bounded illustrative corpus only. It is not intended to mirror the full training or validation corpora used during development.
