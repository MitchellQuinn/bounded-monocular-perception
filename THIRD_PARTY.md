# Third-Party Materials

## Defender 90 Source Asset

The bounded example-image corpus associated with this repository is intended to be derived from a Land Rover Defender 90 source model published on Sketchfab:

- asset: `1997 Land Rover Defender 90`
- source: https://sketchfab.com/3d-models/1997-land-rover-defender-90-8896fdfcb98542d290c04f198766e61c
- publisher: `Ddiaz Design`
- stated license on the source page: `CC BY-NC-SA 4.0`
- Creative Commons license URL: https://creativecommons.org/licenses/by-nc-sa/4.0/

The original `.fbx` source asset is not distributed in this repository.

When the curated Defender example-image corpus is present under [`examples/defender-images`](examples/defender-images), that corpus should be accompanied by the local notice files in that directory and treated according to the terms stated there.
